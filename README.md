# Hybrid C++ Inference Engine

A high-performance, production-ready inference engine in C++ featuring:
- **Dual-Protocol Support:** gRPC (Synchronous) and Apache Kafka (Asynchronous).
- **Deep Learning Integration:** ONNX Runtime (CPU/GPU) with ResNet50 support.
- **Performance Optimizations:** Zero-Copy Memory Pool and Dynamic Batching.
- **Hexagonal Architecture:** Fully decoupled Core from Network Adapters.
